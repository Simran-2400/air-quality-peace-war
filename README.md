## What is CaseIQ?

There's LeetCode for coders. There's nothing like it for business analysts and consultants — until now.

CaseIQ is a free, open-source practice platform where you:
- Read a real business situation (McKinsey-style)
- Submit your approach and recommendations
- Get scored instantly across 5 consulting dimensions
- See exactly what you covered and what you missed
- Learn the underlying framework pattern

**No AI API required. Works offline. Forever free.**

## Scoring System (No API)

Every answer is scored on 5 dimensions (20 points each):

| Dimension | What it measures |
|---|---|
| Problem Framing | Did you identify the core issue? |
| Framework Usage | Did you apply the right mental model? |
| Concept Coverage | Did you cover the essential concepts? |
| Depth of Analysis | Did you hit what consultants prioritise? |
| Recommendation | Is your answer actionable? |
